## Baixando e instalando o projeto
```
git clone https://github.com/cyz/devpira.git
cd [CAMINHO DA PASTA]
npm install
```

### Compilação dos arquivos e hosting local
```
npm run serve
```

### Compilação e compactação para produção
```
npm run build
```

### Execução de testes
```
npm run test
```

### Lints e fixes files
```
npm run lint
```
